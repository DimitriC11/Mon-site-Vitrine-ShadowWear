💡 Conseils

  • Première fois ? Commencez par setup.html pour créer un compte admin
  • Tester l'authentification ? Allez sur test-auth.html pour des tests automatisés
  • Lire la doc complète ? Consultez README_AUTH.md pour tous les détails
  • Données stockées ? Elles sont dans le localStorage du navigateur, persistant entre les sessions
  • Session utilisateur ? Sur sessionStorage, disparaît à la fermeture de l'onglet
  • Plus de comptes à gérer ? Utilisez le panneau admin pour voir tous les utilisateurs
