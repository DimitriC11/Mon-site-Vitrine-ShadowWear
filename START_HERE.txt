⚠️ POINTS IMPORTANTS
════════════════════════════════════════════════════════════════════════════════

1. Les mots de passe sont stockés EN CLAIR dans localStorage
   ➜ Pour la production, utiliser bcryptjs

2. Les données sont LOCALES au navigateur
   ➜ Chaque utilisateur a ses données isolées
   ➜ Pas de synchronisation entre navigateurs

3. Le localStorage PERSISTE même après fermeture
   ➜ Les données restent jusqu'au vidage du cache

4. Le sessionStorage disparaît à la fermeture de l'onglet
   ➜ Utilisé pour la session utilisateur actuelle

A: Admin > "Supprimer TOUT"

Q: Les données persisteront-elles ?
A: Oui, jusqu'au vidage du cache navigateur

A: Oui, chacun aura ses propres données
