Guide d'initialisation du système d'authentification ShadowWear

⚠️ Problème de connexion ?

Vous ne pouvez pas vous connecter car aucun compte utilisateur n'existe encore dans votre navigateur.

Le système utilise le localStorage de votre navigateur pour stocker les utilisateurs. Vous devez d'abord créer les comptes.

📝 Étape 1 : Créer les comptes par défaut

Cliquez sur le bouton ci-dessous pour accéder à la page d'initialisation :

🔧 Initialiser les comptes

Une fois sur la page, cliquez sur "🚀 Créer les comptes par défaut"

✅ Étape 2 : Vérifier les comptes créés

Deux comptes seront automatiquement créés :

👑 Compte Administrateur
📧 Email: admin@shadowwear.com
🔑 Mot de passe: admin123
👤 Compte Utilisateur
📧 Email: user@shadowwear.com
🔑 Mot de passe: user123

🔐 Étape 3 : Se connecter

Maintenant vous pouvez vous connecter avec l'un de ces comptes :

🔑 Page de connexion

✨ Après la connexion

Utilisateur normal : Vous serez redirigé vers account.html (gestion de compte)

Administrateur : Vous pouvez accéder à admin.html (panneau d'administration)

💡 Notes importantes :

• Les comptes sont stockés localement dans votre navigateur

• Chaque navigateur a son propre stockage (Chrome ≠ Firefox)

• Les données persistent même après fermeture du navigateur

• Vous pouvez créer de nouveaux comptes via la page d'inscription
