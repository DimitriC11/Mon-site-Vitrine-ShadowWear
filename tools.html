🧪 Console API

Utilisez ces commandes dans la console du navigateur (F12) :

// Créer des commandes de test
createTestOrders();

// Obtenir les statistiques
OrdersManager.getGlobalStats();

// Les top clients
OrdersManager.getTopClients(5);

// Produits les plus vendus
OrdersManager.getTopProducts(5);

// Toutes les commandes
OrdersManager.getAllOrders();
